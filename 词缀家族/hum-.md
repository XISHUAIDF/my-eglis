| 来源        | 核心含义 | 代表词              | 构成拆解                     | 含义              |
| --------- | ---- | ---------------- | ------------------------ | --------------- |
| **humus** | 土，地面 | **humble**       | hum + ble (adj. 后缀)      | 谦卑的（贴近地面/低姿态）   |
|           |      | **humility**     | hum + ility (名词后缀)       | 谦逊，谦恭           |
|           |      | **humiliate**    | hum + ili + ate (使动)     | 羞辱（把对方按到土里→使丢脸） |
|           |      | **exhume**       | ex (出) + hume (土)        | 挖掘（从土里挖出尸体或真相）  |
|           |      | **inhume**       | in (入) + hume            | 埋葬（放入土中）        |
|           |      | **posthumous**   | post (后) + hum (土) + ous | 死后的/遗腹的（最后入土之后） |
| **homo**  | 人，人类 | **human**        | hum + an (形容词后缀)         | 人的，人类           |
|           |      | **humane**       | hum + ane (形容词后缀)        | 人道的，仁慈的         |
|           |      | **humanity**     | hum + anity (名词后缀)       | 人性，人类，人道        |
|           |      | **homicide**     | homi (人) + cide (杀)      | 杀人，杀人犯          |
|           |      | **Homo sapiens** | homo (人) + sapiens (智慧的) | 智人（现代人种学名）      |


**humdrum**平淡无奇的、平庸的、没亮点的
