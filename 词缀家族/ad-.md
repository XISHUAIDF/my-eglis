| 单词           | 构成                   | 含义演变                      |
| ------------ | -------------------- | ------------------------- |
| **adapt**    | ad + apt (fit)       | 使适合 → 适应                  |
| **adhere**   | ad + haerere (stick) | 黏上去 → 坚持，依附               |
| **admit**    | ad + mittere (send)  | 送去 → 允许进入，承认              |
| **accord**   | ad + cor (心)         | 心靠到一起 → 一致，给予             |
| **affect**   | ad + facere (do)     | 往某处做 → 影响 / 假装（后者：做出一副样子） |
| **aggrieve** | ad + gravare (重压)    | 往身上加压 → 使委屈               |
| **allocate** | ad + locus (place)   | 分派到某个位置 → 分配              |
| **announce** | ad + nuntiare (报告)   | 向……报告 → 宣布                |
| **arrive**   | ad + ripa (河岸)       | 到达岸边 → 抵达                 |
| **assume**   | ad + sumere (拿)      | 拿过来 → 假定，承担               |
| **attend**   | ad + tendere (拉长)    | 把注意力拉向某处 → 参加，照料          |
**ad-** = “向……而去，靠近，附加”。  
它会像变色龙一样，后面的字母被同化，变成 **a-** (在 sc/sp/st 前), **ac-**, **af-**, **ag-**, **al-**, **an-**, **ap-**, **ar-**, **as-**, **at-** 等——只为了让发音更顺口
